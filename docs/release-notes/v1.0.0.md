### 전체 릴리즈 노트 (내부 기록용)

#### 핵심 기능
- 오늘의 카드 뽑기: 매일 1장 타로 카드 + 3D 뒤집기 애니메이션
- 카드 도감: 78장 메이저/마이너 아르카나 전체 열람
- 카드 저장/공유: 워터마크 포함 이미지 갤러리 저장 + SNS 공유

#### 상세 기능
- 일일 고정: 같은 날 재실행 시 동일한 카드 결과 유지
- 정방향/역방향: 50% 확률 적용, 역방향 시 카드 이미지 180도 회전
- 도감 필터: 전체/메이저/완드/컵/소드/펜타클 탭 필터
- 수집 현황: 뽑은 카드는 컬러, 미수집 카드는 그레이스케일 표시
- 올드스쿨 참조: 카드 상세에서 롱프레스 시 라이더 웨이트 원본 모달
- 다크모드: 시스템/라이트/다크 선택
- 매일 알림: 원하는 시간에 카드 뽑기 알림 (기본 09:00)
- 공유 이미지: 9:16 비율, 날짜 포함, "매일타로" 워터마크

#### 기술
- Flutter + Riverpod + Hive + Supabase
- AdMob 배너 + 전면광고 (하루 최대 2회, 첫 실행 시 스킵)
- Firebase Crashlytics 크래시 모니터링
- 오프라인 대응: 로컬 JSON fallback
