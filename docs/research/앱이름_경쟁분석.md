### "Daily Tarot" 키워드 경쟁 강도 평가

- **iOS**: "Daily Tarot" 이름 자체를 쓴 앱이 이미 다수 존재. 검색 결과 상위 1~5위 모두 수천~수만 리뷰를 보유한 기성 앱.
- **Android**: `com.unicorn.dtc`가 "Daily Tarot"이라는 정확한 이름으로 등록됨.
- **결론**: "Daily Tarot"을 앱 이름으로 쓰면 기존 앱들에 묻힐 가능성이 높음. 차별화된 이름이 필요.

## 2. "Today's Tarot" 및 대안 이름 후보 분석

### "Today's Tarot" 현황
- App Store / Google Play 검색에서 정확히 이 이름을 단독 사용한 앱은 미확인 (웹 검색 결과에 등장하지 않음).
- 아직 선점되지 않은 공백 네임스페이스로 보임.

### 대안 이름 후보 7개

| 순위 | 후보명 | 장점 | 단점 | 브랜드 느낌 |
|------|--------|------|------|-------------|
| 1 | **Tarot Petit** | 미니멀·감성 느낌, "소소한 타로"의 프랑스어 뉘앙스, 경쟁 앱 없음 | "petit" 뜻을 모르는 유저에게 의미 불명확 | 고급스럽고 유니크, MZ 감성 |
| 2 | **Tarot Memo** | 기억·저장 기능과 연결, 매일타로의 "오늘 하루 기록" 컨셉과 정합 | "Memo"가 다소 실용적으로 느껴질 수 있음 | 심플, 일상적 |
| 3 | **One Card** | 1일 1카드 컨셉을 이름에서 바로 전달, 미니멀 | 타로 특화 키워드 없어 ASO 불리 | 미니멀, 현대적 |
| 4 | **Tarot a Day** | "매일"의 의미를 영어로 직관적 전달, 검색 친화적 | "a Day" 표현이 다소 평범함 | 명확하지만 평범 |
| 5 | **Today's Tarot** | 아직 선점되지 않은 이름, "오늘"이라는 시의성 명확 | 소유격 표현('s)이 검색 시 변형될 수 있음 | 깔끔하고 직관적 |
| 6 | **Tarot Drop** | 드롭(하루 한 방울) 감성, SNS 브랜드화 가능 | 타로 연결성이 약하게 느껴질 수 있음 | 트렌디, 앱 브랜드 느낌 |
| 7 | **Tarot.me** | 개인화 느낌, "나를 위한 타로" 컨셉 | .me 도메인이 앱 이름으로는 낯설음 | 개인화, 세련됨 |

## 3. 부제(Subtitle) 전략 vs 영어 이름 변경: ASO 관점 비교

### iOS App Store 메타데이터 가중치 (2026 기준)
- **앱 이름(Title)**: 가장 높은 가중치 — 알고리즘의 핵심 랭킹 시그널
- **부제(Subtitle)**: 이름 다음으로 높은 가중치 (30자 제한)
- **키워드 필드**: 부제와 동등한 가중치 (100자 제한)
- **주의**: 동일 키워드를 이름/부제/키워드 필드에 중복 사용하면 자리 낭비

### Google Play 메타데이터 구조
- **앱 이름**: 최고 가중치
- **짧은 설명(Short Description)**: 두 번째 가중치 (80자)
- **긴 설명(Full Description)**: 세 번째 (5000자, 처음 167자 핵심)

### 전략 A: 한국어 이름 "매일타로" 유지 + 영어 부제 추가

```
앱 이름: 매일타로
부제:    Daily Tarot & Card Reading
```

**장점:**
- 한국 스토어에서 "매일타로" 브랜드 독점 (현재 경쟁자 없음, 사용자 확인)
- 부제에 "Daily Tarot" 키워드를 넣어 영어 검색도 커버
- 한국 유저에게 친숙한 이름 유지 → 직접 검색 시 즉시 발견
- 로컬라이제이션: 한국어 로케일에는 한국어 이름, 영어 로케일에는 영어 이름으로 분리 가능

**단점:**
- 글로벌 확장 시 "매일타로"는 한국어를 모르는 유저에게 의미 전달 안 됨
- 영어권에서 앱 이름 자체로는 "daily tarot" 검색 상위 진입 어려움

### 전략 B: 영어 이름으로 완전 교체

```
앱 이름: Today's Tarot (또는 Tarot Petit 등)
부제:    Daily One Card Reading
```

**장점:**
- 글로벌 ASO에서 앱 이름 자체가 키워드를 포함
- 경쟁이 덜한 이름 공간 선점 가능

**단점:**
- 한국 스토어에서 이미 확보한 "매일타로" 브랜드 포기
- 타겟(한국 MZ 여성)에게 한국어 이름이 더 친근하고 바이럴 유리
- "daily tarot" 키워드 자체는 이미 경쟁 강도가 높아 영어 이름으로 바꿔도 상위 진입이 쉽지 않음

### 전략 C: 로케일 분리 (권장)

iOS는 로케일별로 다른 이름을 등록 가능:
- **한국어(ko) 로케일**: `매일타로` + 부제 `하루 한 장 타로 카드`
- **영어(en-US) 로케일**: `Tarot Petit` (또는 `Today's Tarot`) + 부제 `Daily One Card Reading`

이 전략은 한국 시장 브랜드와 글로벌 ASO를 동시에 최적화할 수 있는 가장 유리한 방식.

## 4. 경쟁앱 리뷰 인사이트

### 불만 패턴 (우리가 해결할 기회)
- **Trusted Tarot**: 리딩 저장 및 메모 기능 부재 → "오늘의 카드 저장·공유" 기능이 차별화 포인트
- **Daily Tarot Aura**: 광고 과다, 기능 복잡 → 미니멀 UX + 깔끔한 1카드 경험
- **Labyrinthos**: 기능이 너무 많아 초보자 진입장벽 높음 → 타로 초보도 30초에 끝나는 경험

### 좋아하는 기능 (반드시 포함할 것)
- 카드 의미 설명 (upright/reversed 모두)
- 부드러운 카드 플립 애니메이션
- 매일 새로운 카드 (매일 한 장 뽑기)
- 간단한 yes/no 리딩

## 5. 최종 추천안

### 추천 1순위: 로케일 분리 전략

| 로케일 | 앱 이름 | 부제 |
|--------|---------|------|
| 한국어 | 매일타로 | 하루 한 장 타로 카드 |
| 영어(US) | Tarot Petit | Daily One Card · Minimal |

**근거:**
- "매일타로"는 한국 앱스토어에서 현재 경쟁자 없음 → 브랜드 독점 효과
- "Tarot Petit"은 글로벌 앱스토어에서 선점 가능한 공백 이름
- 프랑스어 "petit(소소한)"은 MZ 감성과 미니멀 컨셉을 함축
- 부제의 "Daily One Card"가 핵심 키워드를 커버

### 추천 2순위: 단순 전략 (런칭 초기)

한국 타겟 MVP 단계에서는 복잡한 로케일 관리 없이:

```
앱 이름: 매일타로
부제:    Daily Tarot · One Card a Day
```

**근거:**
- 한국 시장에 집중하는 MVP 단계에 적합
- 부제에서 "Daily Tarot" 키워드를 확보해 ASO 보완
- 글로벌 확장 시 로케일 분리로 업그레이드 가능

## 6. 키워드 전략 메모

### 한국어 로케일 키워드 풀
- 메인: 타로, 타로카드, 오늘의타로, 매일타로
- 서브: 운세, 오늘운세, 카드점, 타로점

### 영어 로케일 키워드 풀
- 메인: tarot, daily tarot, one card tarot
- 서브: tarot reading, card of the day, tarot card meaning, oracle card

### 중복 제거 원칙
- 이름에 "타로/tarot"가 있으면 부제·키워드 필드에 동일 단어 반복 금지
- 대신 부제는 기능 설명("One Card a Day"), 키워드 필드는 연관어("oracle", "horoscope", "fortune")로 채울 것
